# 手写数字识别案例

## 依赖模块
* os
* json
* gzip
* numpy
* random
* time
* paddlepaddle==2.0.0

## 项目介绍
```
|-datasets: 存放数据和数据读取脚本
    |-mnist.json.gz: mnist数据文件
    |-generate.py: 读取数据的脚本
|-nets: 存放网络结构
    |-fnn.py: 该脚本中定义了单层和多层两种前馈神经网络
|-train.py: 启动训练的脚本
```
